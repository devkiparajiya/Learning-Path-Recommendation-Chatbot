Learning Path Recommendation Chatbot
AI Internship Week 1

This project is a Rule-Based Chatbot developed using Python. The chatbot responds to user inputs using predefined rules and regular expression patterns. It is designed to demonstrate basic chatbot logic and NLP concepts.

Working Principle of this chatbot is as follows:
1.Accepts user input
2.Matches input with predefined regex patterns
3.Returns a suitable response from fixed responses
4.Provides a default response if no rule matches
